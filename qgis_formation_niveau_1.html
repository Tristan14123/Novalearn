Formation QGIS Niveau 1

De Zéro à la Création de votre Première Carte

🎯 Objectifs de Formation

À la fin de ce cours, vous serez capable de :

  • ✅ Installer et configurer QGIS
  • ✅ Comprendre et manipuler les types de données fondamentaux en SIG (vecteur et raster)
  • ✅ Charger, visualiser et styliser des données géographiques
  • ✅ Effectuer des analyses spatiales de base
  • ✅ Créer et exporter une carte de qualité professionnelle

Pourquoi QGIS ?

QGIS (anciennement Quantum GIS) est le leader mondial des SIG open-source. Il est gratuit, puissant, et soutenu par une immense communauté d'utilisateurs et de développeurs. Il n'a aujourd'hui plus rien à envier à ses concurrents payants.

Qu'est-ce qu'un SIG ?

Un SIG est bien plus qu'un simple logiciel de cartographie ; c'est un outil puissant d'analyse, de gestion et de représentation de données qui ont une composante spatiale.

Prêt à commencer votre aventure SIG ?

Module 1 : Prise en Main et Découverte de l'Interface

🎯 Objectifs Pédagogiques

  • • Installer QGIS
  • • Identifier les principaux panneaux de l'interface
  • • Naviguer dans la vue cartographique
  • • Comprendre la notion de "projet"

1.1. Installation de QGIS

La première étape est de télécharger la version la plus adaptée.

  1. 1. Rendez-vous sur le site officiel : qgis.org
  2. 2. Vous verrez plusieurs options. Pour débuter, je vous conseille la version LTR (Long Term Release). Elle est la plus stable, même si elle n'a pas les toutes dernières fonctionnalités.
  3. 3. Téléchargez l'installeur correspondant à votre système d'exploitation (Windows, macOS, Linux) et suivez les instructions.

1.2. L'Écran de Bienvenue : L'Interface de QGIS

Lancez QGIS. L'interface peut sembler complexe, mais décomposons-la :

🔝 Barre de menus

Fichier, Éditer, Vue, Couche, etc. Toutes les fonctionnalités organisées de manière logique.

🔧 Barres d'outils

Icônes pour un accès rapide aux outils les plus courants (enregistrer, zoomer, déplacer, sélectionner...).

📁 Panneau Explorateur

Votre explorateur de fichiers intégré pour naviguer dans vos données SIG.

📚 Panneau Couches

Le cœur de votre projet. Chaque donnée apparaît ici comme une "couche".

🗺️ Canevas de la carte

La zone principale où vos données s'affichent.

📊 Barre d'état

Affiche les coordonnées du curseur, l'échelle actuelle (1:25000), et le SCR du projet.

1.3. La Notion de "Projet"

Un projet QGIS (fichier .qgz) n'est pas la donnée elle-même. C'est un fichier de configuration qui mémorise :

  • • Quelles couches sont chargées
  • • Leur style (couleurs, épaisseurs, étiquettes)
  • • L'emprise géographique de votre dernière vue
  • • Vos mises en page de carte

💡 Analogie Pédagogique

Un projet QGIS est comme une recette de cuisine. La recette (le fichier .qgz) vous dit quels ingrédients utiliser (vos fichiers de données) et comment les présenter, mais elle ne contient pas les ingrédients eux-mêmes.

Si vous déplacez vos fichiers de données, QGIS ne les trouvera plus, tout comme un cuisinier ne peut pas faire son plat si on lui a pris ses légumes !

🏋️ Exercice Pratique 1

  1. 1. Créez un dossier sur votre ordinateur nommé PROJET_QGIS_N1
  2. 2. Ouvrez QGIS
  3. 3. Allez dans Projet → Enregistrer sous...
  4. 4. Naviguez jusqu'à votre dossier et enregistrez le projet sous le nom ma_premiere_carte.qgz
  5. 5. Familiarisez-vous avec les outils de navigation : zoom avant (+), zoom arrière (-), déplacement de la carte (la main)

Module 2 : Les Données en SIG - Le Carburant de vos Cartes

🎯 Objectifs Pédagogiques

  • • Comprendre la différence fondamentale entre les données Vecteur et Raster
  • • Connaître les formats de fichiers les plus courants
  • • Savoir où trouver des données gratuites de qualité

2.1. Le Modèle Vecteur : Le Monde des Objets

Une donnée vecteur représente des objets géographiques par leurs coordonnées géométriques. Il existe trois types principaux :

📍 Point

Représente une localisation unique (un arbre, une borne d'incendie, une ville à petite échelle).

Coordonnées : (x, y)

📏 Ligne

Une série de points connectés. Représente des objets linéaires (rivière, route, sentier).

🔺 Polygone

Une ligne qui se referme sur elle-même. Représente une surface (lac, bâtiment, parcelle).

📋 Table d'attributs

Chaque objet vecteur possède des informations associées, stockées dans une table d'attributs. C'est comme un tableur Excel où chaque ligne correspond à un objet sur la carte et chaque colonne à une information (nom, population, longueur, etc.).

Formats courants :

  • Shapefile (.shp) : Le format historique, très répandu. En réalité, c'est un ensemble de plusieurs fichiers (.shp, .shx, .dbf...)
  • GeoPackage (.gpkg) : Le format moderne, recommandé par QGIS. Un seul fichier qui peut contenir plusieurs couches. Très pratique.
  • GeoJSON, KML : Formats très utilisés sur le web.

2.2. Le Modèle Raster : Le Monde des Pixels

Une donnée raster est une grille (ou une matrice) de pixels, où chaque pixel a une valeur. C'est le modèle des photographies.

📸 Photo aérienne/satellite

Un raster où la valeur de chaque pixel est une couleur.

⛰️ Modèle Numérique de Terrain (MNT)

Un raster où la valeur de chaque pixel représente une altitude.

Formats courants :

  • GeoTIFF (.tif ou .tiff) : Le format le plus classique. Une image TIFF qui contient des informations de géoréférencement.
  • JPEG, PNG : Formats d'image classiques, qui peuvent être géoréférencés avec un fichier "world file" associé (.jgw, .pgw).

💡 Analogie Pédagogique : Vecteur vs Raster

🎯 Le Vecteur

C'est comme un dessin fait avec des instructions : "Trace une ligne de A à B, puis un cercle de centre C et de rayon R". C'est précis, léger et scalable à l'infini sans perte de qualité.

🖼️ Le Raster

C'est comme une photographie. Une mosaïque de points de couleur. Si vous zoomez trop, vous verrez les pixels. Idéal pour des phénomènes continus comme l'altitude ou la température.

2.3. Où trouver des données ?

Pour vos exercices, vous aurez besoin de données. Voici quelques sources fiables et gratuites :

🌍 Natural Earth Data

Données mondiales à petite et moyenne échelle (pays, côtes, rivières...). Parfait pour débuter.

🗺️ OpenStreetMap (OSM)

Base de données mondiale collaborative. Extraction via Geofabrik ou directement via QGIS.

🇫🇷 Portails gouvernementaux

En France : data.gouv.fr et le Géoportail de l'IGN proposent des données de référence (limites administratives, routes, etc.).

Module 3 : Manipulation des Données Vecteur

🎯 Objectifs Pédagogiques

  • • Charger une couche vecteur dans QGIS
  • • Explorer la table d'attributs
  • • Effectuer des sélections simples (spatiales et attributaires)

3.1. Ajouter une couche vecteur

  1. 1. Téléchargez les limites administratives des régions de France depuis data.gouv.fr au format Shapefile
  2. 2. Dézippez le fichier dans votre dossier PROJET_QGIS_N1
  3. 3. Dans QGIS, utilisez le panneau Explorateur. Naviguez jusqu'à votre dossier
  4. 4. Trouvez le fichier regions-20180101.shp et faites-le glisser dans le panneau Couches
  5. 5. 🎉 La carte des régions de France apparaît !

3.2. La Table d'Attributs : Le Cerveau derrière la Forme

  1. 1. Faites un clic droit sur votre couche regions-20180101 dans le panneau Couches
  2. 2. Cliquez sur "Ouvrir la table d'attributs"
  3. 3. Une nouvelle fenêtre s'ouvre avec un tableau

Compréhension clé :

  • • Chaque ligne = une région (un polygone sur votre carte)
  • • Chaque colonne = un champ d'information (nom_region, code_insee, etc.)
  • • Cliquez sur le numéro d'une ligne : la région correspondante est surlignée en jaune sur la carte

3.3. Sélectionner des données

La sélection est une action fondamentale en SIG.

👆 Sélection manuelle

  1. 1. Dans la barre d'outils, activez l'outil "Sélectionner des entités par zone ou simple clic" (icône de curseur jaune)
  2. 2. Cliquez sur une région sur la carte → Elle devient jaune
  3. 3. Maintenez Ctrl pour en sélectionner plusieurs

🔍 Sélection par attributs

Exemple : sélectionner toutes les régions dont le nom commence par "Grand"

  1. 1. Cliquez sur l'icône "Sélectionner des entités par une expression" (icône avec un epsilon ε)
  2. 2. Une calculatrice d'expression s'ouvre
  3. 3. Dans Champs et Valeurs, double-cliquez sur nom_region
  4. 4. Cliquez sur l'opérateur LIKE
  5. 5. Tapez 'Grand %' (le % est un joker)
  6. 6. Expression finale : "nom_region" LIKE 'Grand %'
  7. 7. Cliquez sur "Sélectionner des entités"

🏋️ Exercice Pratique 2

  1. 1. Chargez la couche des régions
  2. 2. Ouvrez la table d'attributs
  3. 3. Sélectionnez manuellement votre région
  4. 4. Utilisez la calculatrice d'expression pour sélectionner toutes les régions dont le code_insee est supérieur à 50

💡 Astuce : Pour l'exercice 4, l'expression sera : "code_insee" > 50

Module 4 : La Symbologie et la Cartographie

L'Art de la Représentation

🎯 Objectifs Pédagogiques

  • • Changer le style d'une couche (couleur, contour)
  • • Créer une symbologie basée sur les attributs (catégorisée, graduée)
  • • Mettre en place des étiquettes
  • • Créer une mise en page simple pour exporter une carte

4.1. Le Panneau de Style

La symbologie est l'apparence de vos couches.

  1. 1. Faites un clic droit sur la couche des régions → Propriétés...
  2. 2. Une grande fenêtre s'ouvre. Allez dans l'onglet Symbologie (l'icône pinceau 🖌️)
  3. 3. Par défaut, vous êtes en "Symbole unique"
  4. 4. Cliquez sur la barre de couleur pour changer la couleur de remplissage ou du contour

4.2. Symbologie Avancée

Le Symbole unique est limité. Explorons mieux :

🎨 Catégorisé

Permet de donner une couleur différente à chaque valeur unique d'une colonne. Parfait pour nos régions.

  1. 1. Changez "Symbole unique" en "Catégorisé"
  2. 2. Dans "Valeur", choisissez le champ nom_region
  3. 3. Cliquez sur "Classifier" → QGIS assigne automatiquement une couleur à chaque région
  4. 4. Cliquez sur "Appliquer" → Votre carte est maintenant bien plus lisible ! 🎉

📊 Gradué

Permet d'appliquer un dégradé de couleur en fonction d'une valeur numérique (ex: densité de population). Le principe est le même, mais on choisit un champ numérique.

4.3. Les Étiquettes

Afficher le nom des régions est essentiel.

  1. 1. Dans la fenêtre Propriétés de la couche, allez dans l'onglet Étiquettes (icône abc)
  2. 2. En haut, changez "Pas d'étiquettes" en "Étiquettes simples"
  3. 3. Dans "Étiqueter avec", choisissez le champ nom_region
  4. 4. Personnalisez la police, la taille, la couleur selon vos préférences
  5. 5. Cliquez sur "Appliquer" → Les noms apparaissent sur la carte ! 🏷️

4.4. La Mise en Page : Créer votre Carte Finale

Le canevas de QGIS est votre espace de travail. Pour créer un document final (PDF, image), on utilise la Mise en page.

  1. 1. Allez dans Projet → Nouvelle mise en page...
  2. 2. Donnez-lui un nom, par ex. Carte_Regions_France
  3. 3. Une nouvelle fenêtre s'ouvre avec une feuille blanche

🗺️ Ajouter une carte

Cliquez sur "Ajouter une carte" et tracez un rectangle sur la feuille. La vue de votre canevas QGIS y apparaît.

📝 Ajouter un titre

Utilisez l'outil "Ajouter un titre" pour donner un nom à votre carte.

🏷️ Ajouter une légende

QGIS génère automatiquement la légende basée sur votre symbologie !

📏 Barre d'échelle & Nord

Ajoutez une barre d'échelle et une flèche du Nord pour compléter votre carte.

💾 Export final

Une fois satisfait, allez dans Mise en page → Exporter comme image... ou Exporter comme PDF...

🏋️ Exercice Pratique 3

  1. 1. Stylisez votre couche des régions en mode Catégorisé par nom
  2. 2. Ajoutez les étiquettes avec les noms des régions
  3. 3. Créez une mise en page contenant :
    • • Votre carte
    • • Un titre
    • • Une légende
    • • Une barre d'échelle
    • • Votre nom comme auteur
  4. 4. Exportez la carte en PNG

🎉 Bravo, vous avez créé votre première carte !

Module 5 : Introduction au Géotraitement

L'Analyse Spatiale

🎯 Objectifs Pédagogiques

  • • Comprendre le concept de géotraitement
  • • Utiliser l'outil "Tampon" (Buffer)
  • • Utiliser l'outil "Découper" (Clip)

Le géotraitement (ou geoprocessing) est l'ensemble des opérations qui permettent de transformer ou d'analyser des données spatiales pour en créer de nouvelles. On les trouve principalement dans le menu Vecteur.

5.1. L'outil Tampon (Buffer)

Cet outil crée un polygone autour d'une entité à une distance spécifiée. Très utile pour définir des zones d'influence ou de protection.

🎯 Scénario

Imaginons que nous voulons visualiser une zone de 50 km autour de la ville de Lyon.

Étape 1 : Créer une couche de points pour les villes

  1. 1. Couche → Créer une couche → Nouvelle couche Shapefile...
  2. 2. Nommez-la villes.shp
  3. 3. Activez le mode édition (icône crayon) pour cette nouvelle couche
  4. 4. Utilisez l'outil "Ajouter un point", cliquez approximativement sur Lyon
  5. 5. Dans la table qui s'ouvre, donnez-lui un nom
  6. 6. Enregistrez les modifications et quittez le mode édition

Étape 2 : Créer le tampon

  1. 1. Vecteur → Outils de géotraitement → Tampon...
  2. 2. Couche source : villes
  3. 3. Distance : 50000 (l'unité est le mètre)
  4. 4. Tampon (en sortie) : Cliquez sur ... → Enregistrer vers un fichier... → zone_influence_lyon.shp
  5. 5. Cliquez sur "Exécuter" → Un cercle de 50 km de rayon apparaît !
